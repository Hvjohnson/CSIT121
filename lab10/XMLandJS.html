Basic information about XML

  • XML is a markup language. It stands for Extensible Markup Language.
  • XML is used to store and transport data.It is designed to be read by humans and machines.
  • XML does not used the same tags as HTML.It is structured in a document tree.
  • XML example:
    book
    title Cat in the Hat title
    author Dr. Seuss author

What is Javascript?

Javascript is an object-based, client-side scripting language that is interpreted by the web browser.

Javascript can be added to HTML and CSS to add different interactive functions; such as changing text or fonts and adding pop up animation.

  • JS uses the DOM, document object model.
  • Object must be declared and properties can be added to the objetcs.
  • You can write to the document using the write() method

Some common elements, properties, methods and functions include:

  • Functions allow you to control when an action is performed. Ex: myFunction() to trigger an alert.
  • Methods: String method charAt(), returns a character in a specified index.
  • Properties are values associated with an object. Ex: Product.name, property is the name.
  • Elements: JS code is embedded with the script element.
