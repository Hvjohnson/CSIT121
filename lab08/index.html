Form Elements: 101

Looking to learn about form elements? You've come to the right place.


Input Element

Creates a text box to allow input of characters. In this example, you can type in your name



Label Element

The label element defines a label for different form elements.

This helps users with screen-readers, as it reads the label out loud.

Select Element

The select element creates a drop down list.This example shows a drop down created with the Select element. It is also made accessible with the label element.

This element allows users to choose from a list of multiple choices in a format that saves space.
